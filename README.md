# AI 100% 활용 게임 개발

## 개요
AI 활용 능력과 생산성을 높이기 위해 클리커 게임을 제작하며

기획, 리소스 제작, 프로그래밍 전 분야에 AI 활용

## 사용 기술
1. Nano Banana(이미지)
2. Grok(애니메이션)
3. ChatGPT(기획)
4. Github Copilot(프로그래밍)

## 개발 기간
2025.10 ~ 2025.11 (2개월)

## 개발 기간
AI에게 온전히 프로그래밍을 맡기는 것은 어렵다고 생각되므로

개발자는 적절히 개입해 생산성을 높이면서도 코드 품질을 유지해야 함
